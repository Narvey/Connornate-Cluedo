f(space, number of moves, seen array, path){
	set current to true in seen array
	add current to path
	if(path has number of moves){
		store current space in targets array  //this array contains the valid move choices.
	}else{
		for (each space in list from adjacency map entry){
			f(space, number of moves, seen array, path)
		}
		remove current from path
		set current to false in seen array
	}
}
/*
Adjacency map (called adjacency list in assignment) is map from integer to linked list of integers

Number of moves does not need to be passed to the function unless you are going to decrement with each recursion.
*/
